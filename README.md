# RiceENCODE-pipeline
The RiceENCODE pipeline test in following dependencies :
Python (2.7) 
samtools (>1.9)
ChIA-PET tools (v3.0)
BWA(0.7.17)
MACS2(2.1.1)
deepTools(2.5.3)
fastp(0.20.0)
cutadapt(1.9.1)
bowtie(1.1.1) for ncRNA-seq or miRNA-seq
TopHat v2.1.0
Cufflinks(2.2.1)
HiC-Pro(2.11.4)
BatMeth2(2.01)
